


# genealogy
## Requirements
The genealogy program requires graphviz and PyQt5

Installing graphviz:

    sudo apt-get install graphviz

Installing PyQt5:

    sudo apt-get install python3-pyqt5

## Running
Run the genealogy program

	python3 application.py


## Tests
Run

	python3 -m unittest tests_tree_env.py
